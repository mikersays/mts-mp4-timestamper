# Product Guide

## Product Overview
MTS to MP4 Video Converter with Timestamp Overlay - A desktop application that converts .MTS video files to .MP4 format while adding a dynamic timestamp overlay showing the original filming date/time.

## Target Users
Professionals who need to archive footage with timestamps, such as:
- Video archivists and documentation specialists
- Legal and compliance professionals requiring timestamped footage
- Corporate media teams managing video archives

## Primary Goal
Enable users to quickly convert MTS files to a more compatible MP4 format with visible timestamps for reference, preserving the original recording date/time as an on-screen overlay that updates as the video plays.

## Key Features
1. **Simple One-Click Conversion** - Minimal configuration required; select a file and convert
2. **Customizable Timestamp Appearance** - Configurable position (corners/center), font size, and visual styling
3. **Batch Processing** - Convert multiple MTS files in a single operation
4. **Dynamic Timestamp** - Overlay updates every minute to reflect actual recording time as video progresses

## Platform Support
- **Windows** - Primary and sole supported platform (aligns with MTS camcorder user base)

## Distribution Model
- **Standalone Executable Package** - No Python or FFmpeg installation required
- Self-contained distribution with bundled FFmpeg binaries
- Ready-to-run application for end users

## User Interfaces
- **GUI Version** - Graphical interface for interactive use
- **CLI Version** - Command-line interface for automation and scripting
